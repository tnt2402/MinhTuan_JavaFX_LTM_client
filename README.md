# appthitracnghiem
Quiz App - Ứng dụng quản lý và cho thi trắc nghiệm

## Công nghệ sử dụng
- Giao diện: Java Swing
- Hệ thống: Java Core, JDBC
- Database: MySQL
- IDE: IntelliJ IDEA Community
